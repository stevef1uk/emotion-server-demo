API Usage

You can find the Docker images on the official DockerHub page, and the source code on our GitHub repository.

Deploy via Docker:

# For Raspberry Pi (32-bit ARM)
docker pull stevef1uk/emotion-emotion-service:arm

# For Apple Silicon (M1/M2/M3)
docker pull stevef1uk/emotion-service:arm64

# For standard x86_64 (Linux/Windows)
docker pull stevef1uk/emotion-service:amd64

# Run the container
docker run -d -p 8080:8080 stevef1uk/emotion-service:arm64

Send a POST request:

curl -X POST http://localhost:8080/predict \
     -H "Content-Type: application/json" \
     -d '{"text": "I‚Äôm so happy to see you!"}'
